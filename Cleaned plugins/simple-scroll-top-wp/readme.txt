A lightweight WordPress plugin that adds a customizable "Back to Top" button with a smooth circular progress ring.

== Description ==

Simple Scroll To Top WP is a lightweight, feature-rich WordPress plugin that adds an elegant "Back to Top" button to your website. The button appears as users scroll down the page and features a smooth circular progress ring that fills as they scroll, providing visual feedback of their position on the page.

This plugin is compatible with all major browsers and works seamlessly with all WordPress themes, including responsive and high-resolution themes.

== Installation ==

= Minimum Requirements =

* WordPress 5.2 or greater
* PHP 7.2 or greater
* MySQL 5.0 or greater

= Automatic Installation =

The easiest way to install Simple Scroll To Top WP:

1. Log in to your WordPress dashboard
2. Navigate to **Plugins > Add New**
3. Search for "Simple Scroll To Top WP"
4. Click **Install Now** next to the plugin
5. Click **Activate Plugin** once installation is complete

= Manual Installation =

1. Download the plugin from WordPress.org or your email
2. Log in to your WordPress hosting via FTP or File Manager
3. Navigate to `/wp-content/plugins/` directory
4. Upload the `simple-scroll-to-top-wp` folder
5. Go to **Plugins** in your WordPress dashboard
6. Find "Simple Scroll To Top WP" and click **Activate**

= After Activation =

1. Go to **Dashboard > Scroll To Top** (in the admin menu)
2. Configure your preferred settings
3. Click **Save Changes**
4. Visit your website and scroll down to see the button in action


== Usage ==

After activating the plugin:

1. Navigate to **Dashboard > Scroll To Top** in the WordPress admin
2. Configure the following settings:
   * **Enabled** — Turn the button on or off
   * **Round Corner** — Choose between circular or square shape
   * **Custom Icon** — Upload a custom icon image (optional)
   * **Button Size** — Set width/height in pixels (30–100 px)
   * **Progress Ring Width** — Set the ring border thickness (1–10 px)
   * **Button Position** — Choose left or right side placement
   * **Primary Color** — Select the button background color
   * **Visibility** — Hide the button on specific device sizes
3. Click **Save Changes**
4. The button will appear on your website when users scroll down

**No additional code or configuration is needed!**


== Frequently Asked Questions ==

= Is this plugin free? =
Yes! Simple Scroll To Top WP is completely free. You can use it on unlimited websites at no cost.

= Will this plugin slow down my site? =
No. The plugin is optimized for performance with minimal CSS and JavaScript. It has negligible impact on page load times.

= Can I customize the button? =
Yes! You can customize:
* Button size (30–100 px)
* Button shape (round or square)
* Button color
* Progress ring width (1–10 px)
* Button position (left or right)
* Custom icon upload
* Visibility per device (mobile, tablet, desktop)

= How do I hide the button on mobile devices? =
In the plugin settings, check the "Hide on Mobile Layout" option under Visibility settings.

= Can I use my own icon? =
Yes! Upload a custom icon in the "Custom Icon" section. Recommended size is 64×64 pixels.

= Does this work with all themes? =
Yes, the plugin is designed to work with all WordPress themes, including custom and responsive themes.

= How do the progress indicators work? =
The circular progress ring around the button fills as users scroll down the page. It starts empty and becomes fully filled when they reach the bottom of the page, giving visual feedback of scroll position.

= Is there a minimum WordPress version? =
Yes, WordPress 5.2 or greater is required. Older versions are not supported.
