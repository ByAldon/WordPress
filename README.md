## Cleaned Plugins

This repository also contains cleaned versions of existing plugins from the WordPress plugin ecosystem.

These modified builds are focused on privacy, security, reduced promotional clutter, and update-locking. They do not unlock paid features, bypass license checks, or remove protections from commercial plugins.

You can find them here:
[Cleaned Plugins](https://github.com/ByAldon/WordPress/tree/main/Cleaned%20plugins)
